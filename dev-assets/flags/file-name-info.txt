The file name format is the following:

  <country>[-<region>][-<ideology>].png

Where
  `country` is the country code (2 chars) e.g. `de` for Germany ()
  `region` is the optional region (any char length) e.g. `by` for an independent Bavaria
  `ideology` is the optional country's ideology (2 chars) which can be:
     `cm` for communism and it's sub-ideologies
     `fm` for fasicsm and it's sub-ideologies
     `dy` for democracy and it's sub-ideologies

Examples:
  German Reich with Black-White-Red flag: de.png
  Nazi Germany with Swastika flag: de-fm.png
  FRG with White-Red-Gold flag: de-frg.png
  GDR with White-Red-Gold with a compass and a hammer encircled with rye: de-gdr.png
  Communist FRG with hammer and sickle: de-frg-cm.png
